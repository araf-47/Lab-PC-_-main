# resp. 1
Since you already know **HTML, CSS, basic JavaScript, and basic Java**, you're in a good position to start Angular. The biggest jump will be learning **TypeScript** and Angular's way of building applications.

Here's a roadmap you can follow:

## Phase 1: Strengthen JavaScript (Very Important)

Before Angular, make sure you're comfortable with:

* Variables (`let`, `const`)
* Functions & Arrow functions
* Objects and Arrays
* Array methods (`map`, `filter`, `reduce`, `find`)
* Destructuring
* Template literals
* Promises
* `async/await`
* ES6 Modules (`import` / `export`)

**Time:** 1 week (if you're already familiar)

---

## Phase 2: Learn TypeScript

Angular is built using TypeScript.

Topics:

* Basic types
* Interfaces
* Classes
* Access modifiers (`public`, `private`, `protected`)
* Generics
* Enums
* Type aliases
* Optional properties
* Union types

**Time:** 1 week

---

## Phase 3: Angular Fundamentals

Start with the basics.

Topics:

* Angular project structure
* Components
* Templates
* Data Binding

* Interpolation
* Property Binding
* Event Binding
* Two-way Binding
* Directives
	  * `*ngIf`
	  * `*ngFor`
	  * `ngClass`
	  * `ngStyle`
* Pipes
* Component communication

  * `@Input()`
  * `@Output()`

---

## Phase 4: Routing

Learn how navigation works.

Topics:

* Angular Router
* Route Parameters
* Child Routes
* Lazy Loading
* Route Guards

---

## Phase 5: Forms

Angular supports two ways of handling forms.

Topics:

* Template-driven Forms
* Reactive Forms ⭐ (most commonly used)
* Form Validation
* Custom Validators

---

## Phase 6: Services & Dependency Injection

Topics:

* Services
* Dependency Injection (DI)
* Singleton services
* Sharing data between components

---

## Phase 7: HTTP & APIs

Topics:

* HttpClient
* GET
* POST
* PUT
* DELETE
* Error Handling
* Loading indicators

Practice by consuming a free REST API like JSONPlaceholder.

---

## Phase 8: RxJS Basics

Angular heavily uses RxJS.

Topics:

* Observables
* `subscribe()`
* `map()`
* `filter()`
* `tap()`
* `switchMap()`
* `BehaviorSubject`

You don't need to master all of RxJS initially.

---

## Phase 9: Advanced Angular

After you're comfortable:

* Standalone Components
* Signals
* Lifecycle Hooks
* Pipes (custom)
* Custom Directives
* Content Projection
* Change Detection
* ViewChild & ContentChild

---

## Phase 10: State Management (Later)

Only after building a few projects.

Learn:

* Services with BehaviorSubject
* NgRx (for large applications)

---

# Projects to Build

Practice is essential. Build progressively:

1. ✅ To-Do App
2. ✅ Calculator
3. ✅ Weather App (API)
4. ✅ Movie Search App
5. ✅ Notes App
6. ✅ Employee Management CRUD
7. ✅ E-commerce Frontend
8. ✅ Authentication (Login/Register)
9. ✅ Admin Dashboard

---

# Suggested Learning Order

1. HTML & CSS ✅
2. JavaScript (ES6+)
3. TypeScript
4. Angular Basics
5. Components
6. Data Binding
7. Directives
8. Routing
9. Forms
10. Services & DI
11. HTTP APIs
12. RxJS
13. Authentication
14. State Management
15. Build Real Projects

### Estimated Timeline

* **Week 1:** JavaScript (ES6+) review
* **Week 2:** TypeScript
* **Weeks 3–4:** Angular fundamentals
* **Weeks 5–6:** Routing, Forms, Services, HTTP
* **Weeks 7–8:** Build 2–3 projects
* **Weeks 9–12:** Advanced Angular and a complete CRUD application

If you study **2–3 hours per day**, you can become comfortable with Angular fundamentals in about **2 months**, and be ready to build real-world applications in around **3 months** with consistent practice.
